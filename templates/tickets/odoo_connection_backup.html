¿Dónde encuentro el master password?

El master password se encuentra en el archivo de configuración de Odoo:

  • /etc/odoo/odoo.conf (Linux)
  • C:\Program Files\Odoo\server\odoo.conf (Windows)

Busca la línea:

admin_passwd = tu_master_password
Formatos de backup:
  • ZIP: Incluye base de datos y filestore (archivos adjuntos). Recomendado para backup completo.
  • SQL Dump: Solo la base de datos PostgreSQL. Más rápido pero sin archivos adjuntos.
