¿Cómo usar esta integración?
  1. Inicia el Servidor: Ejecuta cd whatsapp_server && npm start en una terminal
  2. Conecta tu WhatsApp: Haz clic en "Conectar" y escanea el código QR con tu teléfono
  3. Crea Palabras Clave: Define palabras o frases que activarán respuestas automáticas
  4. Configura Respuestas: Establece qué mensaje se enviará cuando alguien escriba una palabra clave
  5. Ajusta Prioridades: Las palabras con mayor prioridad se evalúan primero
  6. Monitorea Mensajes: Revisa el historial de mensajes y auto-respuestas enviadas
