¿Cómo obtener el Chat ID de Telegram?
Requisito previo: Antes de usar esta función, debes configurar el Bot Token de Telegram en la Configuración del Sistema.

El administrador debe ir a: Configuración → Sistema → Telegram Bot Token
Paso 1: Crear o usar un Bot de Telegram
  1. Habla con BotFather: Busca @BotFather en Telegram
  2. Crea un bot: Envía /newbot y sigue las instrucciones
  3. Guarda el token: BotFather te dará un token como 123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11
  4. Configúralo en el sistema: Pega ese token en la configuración del sistema
Paso 2: Obtener el Chat ID
  1. Busca el bot en Telegram: Busca @userinfobot o @getidsbot
  2. Inicia una conversación: Envía /start al bot
  3. Obtén tu ID: El bot te responderá con tu Chat ID
  4. Para grupos: Agrega el bot al grupo y envía /start@getidsbot
Formato del Chat ID:
  • Chat personal: Número positivo (ejemplo: 123456789)
  • Grupo: Número negativo (ejemplo: -123456789)
  • SuperGrupo/Canal: Número con -100 (ejemplo: -1001234567890)
Importante: Asegúrate de que el bot de tu sistema tenga permisos para enviar mensajes al chat/grupo configurado.
